# Keyboard Shortcuts Reference

Tic-Tac-Stick Quote Engine supports keyboard shortcuts for power users.

## Quick Reference

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Cmd/Ctrl + S` | Save Preset | Save current settings as a preset |
| `Cmd/Ctrl + W` | Add Window Line | Quickly add a new window cleaning line |
| `Cmd/Ctrl + P` | Add Pressure Line | Quickly add a new pressure cleaning line |
| `Cmd/Ctrl + E` | Export to PDF | Generate PDF quote document |
| `Cmd/Ctrl + Shift + C` | Copy Summary | Copy quote summary to clipboard |
| `Cmd/Ctrl + Shift + W` | Window Wizard | Open window wizard dialog |
| `Cmd/Ctrl + Shift + P` | Pressure Wizard | Open pressure cleaning wizard |
| `Cmd/Ctrl + 1` | Toggle Settings | Show/hide job settings panel |
| `Cmd/Ctrl + T` | Focus Title | Jump to quote title field |
| `ESC` | Close Modal | Close any open wizard/modal |
| `?` | Show Help | Display keyboard shortcuts help |

## Toast Notifications

When you use a keyboard shortcut, you'll see a brief toast notification confirming the action. These appear in the top-right corner and disappear automatically.

## Tips

1. **Quick Quote Creation**: Use `Cmd+W` repeatedly to add multiple window lines quickly
2. **Fast Export**: After completing a quote, press `Cmd+E` to export instantly
3. **Wizard Mode**: Use `Cmd+Shift+W` for guided window entry
4. **Help**: Press `?` anytime to see the full shortcut list

## Disabling Shortcuts

If keyboard shortcuts interfere with your workflow, you can disable them by removing or commenting out the `shortcuts.js` script in `index.html`.
