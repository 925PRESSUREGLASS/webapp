## In-App Help System

TicTacStick includes built-in contextual help:

### How to Use

**Help Icons (‚ùì)**
- Look for the small ‚ùì icon next to features
- Click or tap to see helpful information
- Includes tips, examples, and video links

**Help Center**
- Click "Help" button in top right
- Or press `Ctrl + /` (keyboard shortcut)
- Access all documentation
- View keyboard shortcuts
- Contact support

**Contextual Tooltips**
- Hover over UI elements for quick tips
- Automatic help hints for first-time features
- Close with X or by clicking outside
